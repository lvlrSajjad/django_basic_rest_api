# Usage
- Start the server
  - `python3 manage.py runserver`
- Make Migrations 
  - `python3 manage.py makemigrations`
- Migrate 
  - `python3 manage.py migrate`
- Add App/ Module 
  - `python3 manage.py startapp book_api `
